# MParser - C++ 20 Recursive Data Engine.

**MParser** is a lightweight, high-performance data serialization library built with **C++20**. 
It provides a type-safe, JSON-like structure for managing configuration data (or any other data you need), featuring "dotted.path" navigation, 
and built-in encryption modules.
